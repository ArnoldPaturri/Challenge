# Manual de Procedimientos de Monitoreo de Publicidad en Televisión
**Empresa:** TeleAudit Perú S.A.C.  
**Versión:** 2.0  
**Área:** Operaciones – Auditoría de Medios  
**Última actualización:** Julio 2026

---

# 1. Objetivo

Establecer los procedimientos operativos para el monitoreo, auditoría y validación de la publicidad emitida en televisión abierta peruana, garantizando un registro preciso, oportuno y estandarizado de cada emisión publicitaria.

Este manual tiene como finalidad asegurar la calidad de la información entregada a clientes, agencias de medios y anunciantes, permitiendo verificar el cumplimiento de las pautas publicitarias contratadas.

---

# 2. Alcance

Este procedimiento aplica a:

- Auditores de Medios.
- Supervisores de Operaciones.
- Analistas de Auditoría.
- Personal encargado del control de calidad.

Comprende las actividades de:

- Monitoreo en tiempo real.
- Verificación de emisiones.
- Registro de comerciales.
- Identificación de incidencias.
- Validación de horarios.
- Elaboración de reportes.

---

# 3. Clasificación de Televisoras

Para optimizar la asignación operativa, las señales monitoreadas se agrupan de la siguiente manera.

## Grupo TVA

Corresponde a las televisoras comerciales con mayor inversión publicitaria.

| Canal | Descripción |
|--------|-------------|
| América Televisión | Canal comercial nacional |
| Latina Televisión | Canal comercial nacional |
| ATV | Canal comercial nacional |
| Panamericana Televisión | Canal comercial nacional |

### Características

- Alta cantidad de anuncios diarios.
- Bloques comerciales extensos.
- Mayor volumen de campañas nacionales.
- Prioridad alta durante Prime Time.

---

## Grupo TVP

Corresponde a las señales pertenecientes al Instituto Nacional de Radio y Televisión del Perú (IRTP).

| Canal | Descripción |
|--------|-------------|
| TV Perú | Canal estatal principal |
| TV Perú Noticias | Canal informativo |
| Canal IPe | Canal educativo y cultural |

### Características

- Publicidad institucional.
- Campañas gubernamentales.
- Programación educativa y cultural.
- Menor carga comercial.

# 4. Responsabilidades del Auditor

El Auditor de Medios deberá:

- Monitorear continuamente la señal asignada.
- Registrar cada comercial emitido.
- Verificar marca, versión y duración.
- Confirmar el horario de emisión.
- Detectar cualquier incidencia.
- Registrar observaciones objetivas.
- Escalar inmediatamente las incidencias críticas.
- Mantener la confidencialidad de la información monitoreada.

---

# 5. Información que debe registrarse

Cada emisión deberá contener como mínimo los siguientes datos:

| Campo | Descripción |
|--------|-------------|
| Fecha | Día de emisión |
| Canal | Televisora monitoreada |
| Programa | Programa donde aparece el anuncio |
| Hora Inicio | Hora exacta de inicio |
| Hora Fin | Hora exacta de finalización |
| Duración | Duración efectiva |
| Marca | Marca anunciada |
| Producto | Producto o servicio |
| Versión | Duración oficial del comercial |
| Franja Horaria | Según clasificación operativa |
| Observaciones | Información adicional relevante |

# 7. Procedimiento General de Monitoreo

## 7.1 Inicio del Turno

Antes de iniciar el monitoreo, el auditor deberá:

- Verificar el funcionamiento del sistema.
- Confirmar la sincronización del reloj.
- Revisar las señales asignadas.
- Confirmar que las grabaciones estén activas.
- Validar que no existan incidencias pendientes del turno anterior.

---

## 7.2 Durante el Monitoreo

Durante cada bloque comercial se deberá:

1. Detectar el inicio del bloque.
2. Identificar cada comercial.
3. Verificar la marca.
4. Confirmar la versión correcta.
5. Registrar la duración.
6. Registrar la hora exacta.
7. Validar el programa correspondiente.
8. Registrar cualquier anomalía.

---

## 7.3 Finalización del Bloque

Al concluir cada bloque comercial:

- Revisar que no existan registros duplicados.
- Verificar que ningún comercial haya quedado sin registrar.
- Confirmar la duración de todas las emisiones.
- Guardar los cambios en el sistema.

# 8. Tipos de Incidencias

## 8.1 Comercial Cortado (CC)

### Descripción

El anuncio termina antes de completar la duración contratada.

### Ejemplo

Duración contratada:

30 segundos

Duración emitida:

22 segundos

### Código

`INC-CC`

### Acción

- Registrar la incidencia.
- Guardar evidencia.
- Informar al Supervisor.
- Solicitar reposición cuando corresponda.

---

## 8.2 Comercial Incompleto (CI)

### Descripción

El anuncio inicia o termina de forma incorrecta, afectando el contenido del comercial.

### Código

`INC-CI`

### Acción

- Registrar evidencia.
- Informar al Supervisor.

---

## 8.3 Fuera de Franja (FF)

### Descripción

El anuncio se transmite fuera del horario contratado.

### Código

`INC-FF`

### Acción

- Registrar la hora real.
- Registrar la franja contratada.
- Escalar para validación.

---

## 8.4 Omisión de Comercial (OC)

### Descripción

El anuncio no fue emitido durante la pauta programada.

### Código

`INC-OC`

### Acción

- Confirmar la pauta.
- Revisar grabación.
- Reportar inmediatamente.

---

## 8.5 Doble Emisión (DE)

### Descripción

El mismo comercial es emitido más veces de las programadas.

### Código

`INC-DE`

### Acción

- Registrar ambas emisiones.
- Adjuntar evidencia.

---

## 8.6 Error de Versión (EV)

### Descripción

Se emite una versión diferente a la contratada.

### Ejemplo

Contratada:

30"

Emitida:

15"

### Código

`INC-EV`

---

## 8.7 Caída de Señal (CS)

### Descripción

Existe pérdida parcial o total de audio o video durante la emisión.

### Código

`INC-CS`

### Acción

- Registrar hora exacta.
- Registrar duración de la falla.
- Guardar evidencia.
- Escalar inmediatamente.

---

## 8.8 Audio Defectuoso (AD)

### Descripción

El anuncio presenta ausencia de audio, cortes o distorsión.

### Código

`INC-AD`

---

## 8.9 Video Defectuoso (VD)

### Descripción

El anuncio presenta pixelación, congelamiento o problemas visuales.

### Código

`INC-VD`

# 9. Niveles de Prioridad

| Nivel | Tipo de Incidencia | Tiempo de Reporte |
|--------|--------------------|-------------------|
| Baja | Error de registro | Durante el turno |
| Media | Error de versión, fuera de franja | Inmediato |
| Alta | Comercial cortado, doble emisión, omisión | Inmediato |
| Crítica | Caída de señal o pérdida de grabación | Comunicación inmediata |

---

# 10. Criterios de Calidad

Todo registro deberá cumplir con los siguientes principios:

- Exactitud.
- Objetividad.
- Integridad.
- Consistencia.
- Trazabilidad.
- Puntualidad.

No deberán eliminarse registros sin autorización del Supervisor de Operaciones.

---

# 11. Buenas Prácticas

- Mantener atención permanente durante el monitoreo.
- No registrar comerciales sin evidencia.
- Confirmar siempre la duración del anuncio.
- Revisar cuidadosamente las distintas versiones de una campaña.
- Evitar registros duplicados.
- Documentar cualquier situación extraordinaria mediante observaciones.
- Mantener comunicación constante con el Supervisor.

---

# 12. Escalamiento de Incidencias

Cuando se detecte una incidencia crítica, el auditor deberá seguir el siguiente flujo:

```text
Detección
      │
      ▼
Registro en el Sistema
      │
      ▼
Validación del Auditor
      │
      ▼
Notificación al Supervisor
      │
      ▼
Evaluación de Evidencia
      │
      ▼
Reporte al Cliente (si corresponde)
```
